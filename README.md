# Teslo-shop API

1. Clonar proyecto
2. 
```
npm installl
```
3. Clonar el archivo ```.env-template``` y renombralo ```.env```
4. Cambiar variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
6. Ejecutar SEED
```
http://localhost:3000/api/seed
```
7. Levantar proyecto: 
```
npm run start:dev
```
